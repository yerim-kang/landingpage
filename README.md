# 빵삼폰 Bbangsam Landing Page

> 알뜰폰 요금제 & 인기 스마트폰 특가를 소개하는 **프로모션 랜딩 페이지**입니다.

## 🔗 Demo
- Live Site: https://bbangsamlandingpage.netlify.app/  
- GitHub: https://github.com/yerim-kang/landingpage

---

## 🔥 Concept

빵삼텔레콤의 휴대폰 특가 & 알뜰 요금제 혜택을  
**한 페이지에서 빠르게 보고 결정**할 수 있도록 만든 랜딩 페이지입니다.  
단독 특가 + 간편 구매 안내 + 후기 중심으로 구성되어 있습니다.

---

## 📌 Main Features

- **요금제 & 유심/셀프가입 정보** 섹션  
- 인기 스마트폰 **특가 모델 리스트**  
- 간편 **구매 진행 절차 안내**  
- 현재 진행 중인 **이벤트 & 혜택 소개**  
- 고객 **리뷰 후기 섹션**  
- 고객센터 연락 안내
  
---

## 🧠 Tech Stack

- HTML / CSS / JavaScript / jQuery
- 사용 라이브러리/플러그인 : AOS, Swiper
- Deploy: Netlify
